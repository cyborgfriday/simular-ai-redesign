→ Before You Try

The prototype is a realistic simulation of the redesigned SAI interface. Here are a few things to know before jumping in.

  • Use Approve Plan, Modify Plan, and Sign In buttons to advance states.
  • Covers task submission, plan review, mid-task modification, and auth handoff. Execution itself is not simulated.
  • Some interactions are locked — intentional, not broken.

Suggested flow: Submit a task → Review the execution plan → Modify a step → Approve → Handle the sign-in handoff.
